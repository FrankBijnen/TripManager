Command line parameters
TripManager can be started with commandline parameters, to postprocess, or create files from a GPX. The intended use is in CMD, or BAT files. Use /? to display the supported options.

All settings interactively made will be used.  This includes the model and the Trip option. 6) Send to device

C:\Program Files\TripManager>tripmanager /?

C:\Program Files\TripManager>

Usage: C:\Program Files\TripManager\TripManager.exe /Options GPX files Mask

Options:
        /PP or /PostProcess = Postprocess
        /Trips              = Create .trip Zumo Trips
        /Routes             = Create GPX Stripped Routes
        /Tracks             = Create GPX Tracks
        /Wpts or /WayPoints = Create GPX WayPoints
        /Poi or /Gpi        = Create Points Of Interest
        /Kml                = Create KML Google Earth
        /Html               = Create HTML
        /Poly               = Create POLY

Example: C:\Program Files\TripManager\TripManager.exe /PP /Tracks /Trips *.gpx
