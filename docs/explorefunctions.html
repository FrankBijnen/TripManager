Explore functions

The Explore functions are only enabled when Enable Explore is set to True in Advanced settings/Device settings, and the connected device (XT2/ Tread 2) contains an Explore.db

Query   
Opens the table browser, of the selected DB on the Temp folder. (Expert option, intended use is for debugging)

Compare Explore.db with Trips
Compare the content of the Explore.db and compares it with the Trips in the Trips folder. (Expert option, intended use is for debugging)
Only useful for an XT2, or Tread 2, to find inconsistencies, that may have been introduced by manually copying, deleting, or renaming trip files.

Advice when using TripManager in conjunction with Explore.
To prevent inconsistencies it is recommended to:
  • Delete trips on the device, and not from the Trips folder.
  • A route initially send using the normal 'Garmin' procedure, will be stored as '1234567890.trip'. If you send the same route again using TripManager it will be stored using the <route_name>.trip.  This will lead to duplicates. In this case delete the trip from the device first, to avoid inconsistencies.
