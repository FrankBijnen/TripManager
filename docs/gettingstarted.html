Getting started

 Previous  Next
  • Install the program via the installer. That will create a Desktop icon if requested.
  • Make the .System folder visible on your zumo. https://www.zumouserforums.co.uk/app.php/ZXT-P84
  • Verify that the .System\Trips folder exists. If the Zumo has just been reset to factory defaults, it may be missing. The folder will be created when a trip/route is saved using the Trip planner, or by importing a GPX containing a route.
  • Connect the Zumo, and wait for the device to appear in Windows Explorer.
  • Start the program. If a supported device is connected it will be selected automatically.
  • Starting with V1.5 TripManager detects connecting and disconnecting of the Zumo.
