Overview


TripManager has 4 windows, that you can resize as needed.
  • Device file list (Top Left)
  • Windows file list (Top Right)
  • Trip Window (Bottom Left)
  • Hex/OSM Window (Bottom Right)

Main menu

Help/About
Shows the Version.

Help/documentation
Opens a link to the Installed documentation (if selected during install)
Opens a link to the Online documentation

Advanced/Settings
Opens a dialog where you can view/edit TripManager settings.
  • General
  • Device settings
  • Zumo settings
  • GeoCode settings.
  • Enabling GeoCoding needs to be done on this tab, where you should enter an Api_key
