Using the clipboard
From Excel to TripManager.
  • Prepare a sheet in Excel with the address in column E. No column heading needed. Address is the minimal info needed. Other columns can be added if desired and available.

Tip: if you enter data in columns C,D (Lat,Lon) set the format In Excel to text and enter values in decimal form. For an example see From Tripmanager to Excel
  • Select all lines, Columns A-E, and copy to Clipboard (CTRL+C)
  • Create a new empty Trip in TripManager See 12 Using Trip Editor
  • In TripManager Right Click/Paste, or CTRL+V
  • Because columns A-D were empty in Excel default route point names are assigned, all points are Shape, and there are no Coordinates.
  • You can now look up the addresses to obtain the coordinates. See Looking up addresses
  • When done save the trip, or export to GPX/CSV.


Note: The example above copies all route points at once. Copying a selection as a block, and inserting after an existing point is also supported. Here RtePt 5 to 7 are copied and inserted after RtePt 11.



From Tripmanager to Excel

  • Create a new empty sheet in Excel and set at least column C and D as text.



  • Select all lines in TripManager (CTRL+A) and copy to clipboard (CTRL+C)



  • Paste in Excel.
  • Do with it whatever you like.

Using CSV files
Prepare a CSV file
  • The CSV should contain a heading line with the column names. These column names are recognized. Route point;Point type;Lat;Lon;Address
  • Tip: The column names are the same as in TripManager except ID, that can not be used.
  • The CSV should contain at least 1 column: Address.
  • Columns Lat/Lon are not mandatory, but if you supply them use the decimal separator as specified in your Windows regional settings! (usually a comma, or a point)

From CSV to TripManager
  • Create a new empty Trip in Tripmanager.
  • Click on the CSV button, Import.


  • Open the prepared CSV file. The result should look like this. You can now look for addresses.



Note: If the point type is not supplied it is set to Via.

From TripManager to CSV
  • Creating a CSV file is as simple as opening a trip in TripManager and clicking on CSV/Export.

Note: You can use the created CSV file for importing.
Looking up addresses


Select the lines to look up, or select all lines (CTRL+A) and click on the Search button.

When no coordinates are available TripManager will search for the coordinates from the address.
If coordinates are available TripManager will search for the place.

For every selected line the search dialog will be shown. Select the matching line and accept the proposed coordinates with OK.

Cancel will not update the Coordinates, Retry allows you to change the search parameter and retry.


Note: While the trip has not been calculated, TM will only draw straight lines between the route points.
